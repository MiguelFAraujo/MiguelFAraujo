## Infraestrutura Self-Hosted

Cluster caseiro com containers Docker rodando 24/7 em Raspberry Pi 4 + Orange Pi Zero 2W, conectados via NFS e Tailscale.

Servicos principais: Nextcloud (nuvem pessoal), Gitea (Git self-hosted), AdGuard Home (DNS), n8n (automacoes), OmniRoute (AI Gateway), Portainer, Duplicati (backup), Calibre (ebooks).
